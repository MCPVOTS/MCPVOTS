## üöÄ Getting Started

### Prerequisites

- Node.js 18+
- Python 3.11+
- Git

### Installation

```bash
# Clone the repository
git clone https://github.com/kabrony/MCPVots.git
cd MCPVots

# Install dependencies
npm install

# Start the development server
npm run dev
```

Access the platform at `http://localhost:5173`.
